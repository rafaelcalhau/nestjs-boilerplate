# nestjs-boilerplate
Just a boilerplate to build amazing backend apps with NestJS.


## Using with Docker

Install and run at port 3000 with --watch on:
```bash
$ docker-compose up -d
```

## Not using docker? Running the app

```bash
$ npm install

# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
